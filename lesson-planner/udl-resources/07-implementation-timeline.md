# UDL Implementation Timeline

UDL Implementation doesn't happen all at once. Slow and steady wins the race!

## Basic Tips for UDL Implementation

- It will take some time for all educators to get on board
- Typical implementation is a 4-6 year process
- Year one is often a chance for "early adopters" to get on board
- "Full implementation" is reached once 50% of team members are using UDL in their daily practice and it is influencing student outcomes
