## Overview
This directory contains 150+ thinking routine PDFs from Harvard Project Zero, scraped from https://pz.harvard.edu/thinking-routines-all

## Directory Structure
- `pdfs/` - Original PDF files
- `txt/` - Extracted text files
- `pdf_links.txt` - List of all PDF URLs
